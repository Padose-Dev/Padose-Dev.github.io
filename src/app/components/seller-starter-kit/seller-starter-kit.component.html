How to Write Descriptions

Structure your description in this order for maximum clarity.

1
What is it?

One sentence about the product

2
Key features

3-5 bullet points: material, size, weight, specs

3
Who is it for?

Mention the use case or occasion

4
Care instructions

If applicable (wash, store, maintain)

Example Description

Handwoven Kanchipuram silk saree with traditional zari border and pallu.

  • Pure mulberry silk, 6.2 meters with blouse piece
  • Weight: 650g (heavy feel, premium drape)
  • Zari: Genuine copper-gold thread work
  • Suitable for weddings, festivals, and special occasions

Dry clean only. Store folded in muslin cloth.

Quick Specs Reference

Images
  • Format: JPG, PNG, or WebP
  • Max width: 1200 px
  • Store: 1 main + 1 logo + 3 banners
  • Products/Services: up to 10 each
Videos
  • Format: MP4 (recommended)
  • Max length: 60 seconds
  • Max size: 25 MB
  • Up to 3 per store/product/service
Variants
  • Products: name + price (min 1)
  • Services: name + duration + price
  • Keep names short and consistent
  • One attribute per variant
